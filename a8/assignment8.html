Player Profile

API Information

Description: This API includes statistics and profiles of many soccer players around the world.

Website: The website I found it on is https://www.api-football.com.

Parameters Used: I used the "player" parameter to find the specific player in the API, which took the player's unique ID number to index the profile.

Applications: This API might be useful for a sports website like ESPN, or for a video game like FIFA to display statistics about certain players.
